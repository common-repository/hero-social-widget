== Description ==

Display your social profiles in a cool widget.

You have the following social profiles options:
-Facebook
-Twitter
-Google Plus
-Flickr
-RSS
-DeviantArt
-Youtube
-Vimeo
-Dribbble
-LinkedIn
-Pinterest
-Behance

== Installation ==

Pretty simple. Just follow this steps

1. Upload folder `hero-social-widget` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Appearance -> Widgets, and drag the Hero Social Widget to any sidebar where you want to display your social profiles.
